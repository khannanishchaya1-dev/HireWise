# HireWise 🚀

HireWise is an AI-powered interview preparation platform that helps candidates improve their chances of getting hired by providing resume analysis, skill-gap detection, interview preparation tools, and ATS-friendly resume optimization.

## 🌟 Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* HTTP-Only Cookie Based Authentication
* Protected Routes
* Logout Functionality
* User Context Management

### User Dashboard

* Personalized Homepage
* Profile Management
* Authentication Persistence
* Secure Session Handling

### AI Features

* Resume Analysis
* Skill Gap Detection
* Job Description Matching
* AI Interview Question Generation
* ATS Resume Optimization
* Progress Tracking

## 🔐 Authentication Flow

```text
User Login
     ↓
JWT Generated
     ↓
Stored in HTTP-Only Cookie
     ↓
Browser Sends Cookie Automatically
     ↓
Backend Middleware Verifies JWT
     ↓
Access Granted
```
